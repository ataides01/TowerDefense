🏹 Tower Defense PRO

Um jogo de Tower Defense desenvolvido com HTML, CSS e JavaScript puro, focado em mecânicas clássicas de estratégia em tempo real, com sistema de progressão, inimigos dinâmicos e batalhas contra chefes.

🎮 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos de:

Lógica de programação
Manipulação do DOM
Animações com Canvas
Estruturas de dados em jogos
Game loop e física básica

O jogo simula um sistema clássico de defesa de base, onde o jogador deve posicionar torres estrategicamente para impedir que os inimigos alcancem o final do caminho.

Evitar que os inimigos cheguem ao final do caminho, utilizando estratégia para posicionar torres e sobreviver o maior tempo possível.
